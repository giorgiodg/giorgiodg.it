# giorgiodg.it

## Giorgio Delle Grottaglie's one-page website

For the first time ever I built a personal website for myself, and here's the source code, just in case you're curious to know how it works.  
A big thanks to Konstantin Münster who inspired me with his awesome [minimal portfolio starter](https://github.com/konstantinmuenster/gatsby-starter-portfolio-minimal).

## Technologies / frameworks I am using

- Gatsby
- ReactJS
- cypress.io

## How to run it

1. **Install the Gatsby CLI.**

   `npm install -g gatsby-cli`

2. **Create a new Gatsby site with the Portfolio Minimal starter.**

   `git clone https://github.com/giorgiodg/giorgiodg.it`

3. **Start the site in `develop` mode.**

   `cd giorgiodg.it`  
   `gatsby develop`
